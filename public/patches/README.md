# Patch art

Drop the embroidered patch PNGs in this folder. The leaderboard and player
cards reference these exact filenames:

- `sweep.png` — Top Dawgs SWEEP / "Clean Sweep!" patch
- `mini-sweep.png` — Top Dawgs MINI SWEEP / "Bickford Patch" patch

Square art (1024×1024 or 512×512) on a transparent background works best.
Files in this folder are served from `/patches/<filename>` at runtime.
